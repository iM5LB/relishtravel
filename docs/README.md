## What is RelishTravel?

RelishTravel transforms the Elytra flight experience by adding:
- **Charge-based launching** - Sneak + Jump, land to charge, release to launch
- **Mid-air boosting** - Sneak while gliding for speed bursts
- **Safety features** - Damage prevention and obstruction detection
- **Visual feedback** - Particles, sounds, and action bar displays
- **Permission tiers** - Configurable boost limits per rank
